Decision Fusion for Ear-Drum Abnormality Detection

Decision fusion framework for detecting middle ear infections using otoscopic image analysis (VGG19) and tympanometry data classification (Random Forest).

Overview

Middle ear infections (Otitis Media) are commonly diagnosed using otoscopic inspection and tympanometry tests. Each diagnostic method has limitations when used independently.

This project introduces a decision fusion system that combines:

Deep learning for image analysis
Machine learning for tympanometry classification
The combined system improves diagnostic reliability and accuracy.

Dataset

The model uses two types of data:

Otoscopic Images

Images of the ear canal used for detecting abnormalities.

Tympanometry Data

Clinical measurements describing eardrum movement and pressure variations.

Model Training

Example code for training the VGG19 model:

from tensorflow.keras.applications import VGG19
from tensorflow.keras.layers import Dense, Flatten
from tensorflow.keras.models import Model

base_model = VGG19(weights='imagenet', include_top=False)

x = Flatten()(base_model.output)
x = Dense(256, activation='relu')(x)
output = Dense(1, activation='sigmoid')(x)

model = Model(inputs=base_model.input, outputs=output)

model.compile(
    optimizer='adam',
    loss='binary_crossentropy',
    metrics=['accuracy']
)

Decision Fusion

The final diagnosis is generated by combining predictions from both models.

Example fusion logic:

def decision_fusion(image_pred, tymp_pred):

    if image_pred == 1 or tymp_pred == 1:
        return "Infection Detected"

    return "Normal"
